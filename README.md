# 📸 Photography Event Booking Website

A fully responsive **Photography Event Booking Website** built using **HTML, CSS, and JavaScript**. This website allows users to explore photographer details, services, sample work, pricing packages, and also book events through contact and payment forms.

---

## ✨ Features

### 👤 About Photographer

* Displays expertise and specialization.
* Professional introduction and background.

### 🎯 Services Offered

* Wedding Photography
* Event Coverage
* Portrait Shoots
* Product Photography
* Custom Shoot Options

### 🖼️ Sample Work

* Image gallery showcasing real samples.
* Video sample links stored in `video-links.md`.

### 💰 Packages & Pricing

* Clear package details and pricing.
* Users can compare and choose best options.

### 📬 Contact Form

* Users can submit inquiries or booking requests.

### 💳 Payment Page

* Custom payment UI (`payment.html`).
* Styled using `payment.css` and functions handled via `payment.js`.

### 📱 Fully Responsive Design

* Works smoothly across mobile, tablet, and desktop.

---

## 🛠️ Tech Stack

| Layer    | Technology              |
| -------- | ----------------------- |
| Frontend | HTML, CSS, JavaScript   |
| Styling  | styles.css, payment.css |
| Scripts  | script.js, payment.js   |
| Assets   | images/, videos/        |
